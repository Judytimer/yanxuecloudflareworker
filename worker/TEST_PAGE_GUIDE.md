# 测试页面使用指南

## 访问测试页面

### 方法 1: 使用本地服务器（已启动）

**访问地址：**
```
http://localhost:8080/test.html
```

在浏览器中打开上述地址即可使用测试页面。

### 方法 2: 直接打开文件

如果本地服务器未运行，可以直接在浏览器中打开文件：
```
文件路径: /home/judytimer/cloudflare/worker/test.html
```

右键点击文件 → 选择"用浏览器打开"

## 测试页面功能

### 1. 基本查询测试
- 点击"测试基本查询"按钮
- 测试 GraphQL 连接是否正常

### 2. 发送消息
- 在"输入消息内容"框中输入你的问题
- （可选）输入会话ID
- 点击"发送消息"按钮
- 查看 AI 回复

### 3. 快速测试按钮
- "快速测试：你好" - 测试中文问候
- "快速测试：AI问题" - 测试 AI 相关问题
- "快速测试：英文" - 测试英文对话
- "快速测试：数学" - 测试数学问题

## 停止服务器

如果需要停止本地测试服务器：

```bash
# 方法 1: 使用保存的 PID
kill $(cat /tmp/test_server.pid)

# 方法 2: 查找并停止进程
pkill -f "python3 -m http.server 8080"
```

## 故障排查

### 问题 1: 无法访问测试页面
- 检查服务器是否运行：`ps aux | grep "http.server"`
- 检查端口是否被占用：`lsof -i :8080`
- 尝试使用其他端口：`python3 -m http.server 8081`

### 问题 2: 测试失败
- 检查 Worker 端点是否可访问
- 查看浏览器控制台（F12）的错误信息
- 确认网络连接正常

### 问题 3: CORS 错误
- Worker 已配置 CORS，应该不会有问题
- 如果出现 CORS 错误，检查 Worker 配置
